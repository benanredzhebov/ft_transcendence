# ft_transcendence

## how to run

```
cd ./frontend
npm install
npm run dev

npm install react react-dom
npm install --save-dev @types/react @types/react-dom

npm install socket.io-client
npm install --save-dev @types/socket.io-client


cd./backend
npm install socket.io
```
